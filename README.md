# Unraid Community Apps Templates

These are Unraid Community App templates for the following apps:

## MCSManager (Web/Daemon)
The web panel is a lightweight, open-source, web-based game server management panel. Supports Minecraft Java/Bedrock, Steam games, real-time logging, automated backups, and multi-server management. The daemon is the backend component that manages and runs game server instances. Works alongside the MCSManager-Web template. Supports Minecraft Java/Bedrock and most Steam game servers. Both the daemon and web panel are required in order to use the tool fully.

These XML files are based on the official MCSManager docker file: https://github.com/MCSManager/MCSManager?tab=readme-ov-file#docker-installation

Support thread for the CA template: https://forums.unraid.net/topic/197644-support-mcsmanager-daemon-and-mcsmanager-web

## FleetDM
An open-source, web-based device management solution. Allows you to manage macOS, Windows, Linux, iOS, Android, and Chromebook devices remotely.

This XML file is based on the official Fleet docker deployment instructions: https://fleetdm.com/guides/deploy-fleet-on-docker-compose

Support thread for the CA template: https://forums.unraid.net/topic/198204-support-fleet-dm/
