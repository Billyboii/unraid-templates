# Unraid Community Apps templates

These are Unraid Community App templates for the following apps:

- MCSManager (MCSM)
- FleetDM

These are Unraid templates made by the Unraid community for deployment. These templates are not currently associated with any members of the development teams for the associated projects.

## MCSManager
This XML file is based on the official MCSManager docker file: https://github.com/MCSManager/MCSManager?tab=readme-ov-file#docker-installation
## FleetDM
This XML file is based on the official Fleet docker deployment instructions: https://fleetdm.com/guides/deploy-fleet-on-docker-compose
